# FinPlan

FinPlan is an Angular + Firebase app for tracking progress toward a house purchase. The current product includes authenticated access, a dashboard goal view, source breakdowns, expected funds, activities, a separate currency tracker, and English/Ukrainian localization.

The implementation-aligned product documentation lives in [docs/CURRENT_FUNCTIONALITY.md](docs/CURRENT_FUNCTIONALITY.md).

## Scripts

```bash
npm start
npm run build
npm run test
npm run lint
```

## Main Stack

- Angular 21
- TypeScript
- Firebase Auth
- Firestore
- Angular Material
- Chart.js
- Vitest
